EXERCÌCIO PARA NOTA 2

Elaborar uma página HTML que represente um formulário, com campos de entrada, para um cadastro de pessoa jurídica, com os seguintes campos:

1- CNPJ (só números permitidos) - caixa de texto, 14 caracteres max
2- Email - caixa de texto, formato de email
3- Razão social - caixa de texto , 60 caracteres max
4- Tipo de empresa - combo box (privada, pública, ong, adm direta)
5- Indicador de inserção fiscal - radio button (isenta, não isenta)
6- Modalidades de contratação - check box (CLT, RJU, autônomo, PJ terceirizada)
7- Faturamento anual (máscara de $$) - caixa de texto (é possível ter a máscara só com HMTL?)
8- Uma área de texto livre (5 linhas, 15 colunas) para descrição das atividades da empresa.
9- Site da empresa - caixa de texto, formato de URL.
10- Opção para realizar o upload de um arquivo, que contém o contrato social.
11- Telefone da empresa - caixa de texto, formato de telefone.
12- Um botão de submissão do formulário para uma URL de serviço.
13- Um botão para limpar os campos (é possível ter a funcionalidade de limpar só com HTML?)

- A página deve ter um título e um cabeçalho.
- A página deve ter os textos definidos por tags "label".
- Os textos e campos devem ser alinhados por tabelas.
- A página deve ser hospedada em um servidor web (local ou remoto).
- Deve ser escrito um serviço simples (a ser executado no backend), que será acionado pela url indicada no formulário. Este serviço deve simplesmente imprimir no console todos os dados recebidos do formulário. 
- O serviço em questão deve ser disponibilizado no mesmo servidor web da página.
